# Assignment: IT in Different Industries

**Type:** Individual Assignment

---

## Description

For this assignment I chose three companies from different industries and explored how the integration of information technology has advanced each company's operations, profits, and organization. The three companies I selected were:

- **Amazon** (Retail / E-Commerce & Logistics)
- **Allegheny Health Network** (Healthcare)
- **Tesla** (Automotive / Manufacturing)

Each company represented a different way that technology can fundamentally transform how an organization operates — from warehouse robotics and AI documentation tools to vertically integrated software manufacturing.
