# L02: Binary Code Exploration

**Module:** L02  
**Points:** 10  
**Type:** Individual Assignment

---

## Description

In this activity I explored binary representation through self-paced practice. I watched a video on binary representation, used an ASCII table to manually translate binary strings into readable text, and used an online binary converter tool to verify my answers. The three strings translated to "Hello", "This is a binary code", and "I love data."

## Reflection

This was one of the more satisfying early assignments because it connected directly to something I already had a passing interest in. I had heard about binary my whole life around computers but never actually sat down and worked through the translation myself. Using the ASCII table manually before checking with a converter gave me a real feel for how the system works rather than just getting an answer. It reinforced just how foundational binary is to everything we interact with digitally every single day.
