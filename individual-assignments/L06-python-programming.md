## Description

For this assignment I completed the first three modules of the Grok Academy "Disease Epidemic" activity to learn the basics of Python programming. Each module built on the last, culminating in calculating the rate of "Green Glow" spread. I took screenshots of each printed output and submitted them as documentation of my work.

## Reflection

This was one of the more personally relevant assignments since I was already taking IST 140 at the same time and actively learning Python. Seeing a completely different platform approach the same language reinforced concepts I was working through in my other course. The Disease Epidemic framing was a clever way to make what could have been dry syntax practice feel more engaging. It also gave me early exposure to print statements and variables in a context that made their purpose immediately obvious.
