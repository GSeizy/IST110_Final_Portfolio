# L07: SQL

**Module:** L07  
**Points:** 12  
**Type:** Individual Assignment

---

## Description

In this assignment I used the W3Schools Online SQL Editor to practice writing basic SQL statements against a sample database containing Customer and Categories tables. I wrote and ran SELECT, INSERT, and DELETE statements including filtering with WHERE, sorting with ORDER BY, and removing duplicates with SELECT DISTINCT. Screenshots of each query and its output were compiled into a Word document and submitted.

## Reflection

SQL was one of the topics this semester that I knew I would need to understand for cybersecurity, so this assignment felt especially purposeful. Being able to query a database is a fundamental skill whether you are building something, auditing something, or looking for vulnerabilities. The W3Schools editor made it low-friction to experiment and see results immediately, which is a great way to learn syntax. Writing the DELETE statement in particular drove home how much power a single line of code can have over a dataset.
