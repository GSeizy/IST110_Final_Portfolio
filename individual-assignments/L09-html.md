# L09: HTML

**Module:** L09  
**Points:** 10  
**Type:** Individual Assignment

---

## Description

For this assignment I used the W3Schools Tryit Editor to practice working with HTML. I ran sample code for five core HTML tags including documents, headings, paragraphs, links, and images, then built a basic webpage that included a heading, body text, an embedded link, and a picture. A screenshot of the Tryit Editor showing both the code and the rendered output was submitted as the deliverable.

## Reflection

Coming into this course from IST 250 where I had already built full websites, this assignment was familiar territory. But it was a good reminder that the fundamentals matter. HTML is the backbone of everything on the web and understanding how each tag contributes to the structure of a page is something that is easy to take for granted once you get comfortable with it. It also reinforced my appreciation for how approachable web development actually is once you get past the initial learning curve.
