# L09: JavaScript

**Module:** L09  
**Points:** 12  
**Type:** Individual Assignment

---

## Description

For this assignment I used the W3Schools JavaScript tutorial to practice basic JavaScript concepts. I worked through window.alert() output, JS Events by creating a button that displayed the current time with my name, and JS Conditions by comparing if, else, and else if control structures. Screenshots of each output were compiled with written answers comparing the examples and submitted as a document.

## Reflection

JavaScript was one of the first languages I learned in IST 250, so revisiting it here felt natural. The comparison between event-driven JavaScript and the conditional control structures was a useful exercise in thinking about when and why code executes. The contrast between code that runs immediately and code that waits for user interaction is a concept that comes up constantly in web development and programming in general. It also tied back nicely to the Python work I was doing simultaneously in IST 140.
